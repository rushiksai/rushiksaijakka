Projects

Autonomous Delivery Robot

ROS Python LiDAR Gazebo Rviz

Laser scan data checks for the door in front of the robot. The distance to the door is checked. Once the distance increases over a predetermined threshold, the robot initializes movement. Before moving, the robot waits until the door is fully opened. Movement is stopped if there is an obstacle about 0.5m in front of the robot (e.g. the robot reaches the wall in the next room). Visualization: laser scan data in rviz shows the door being opened.

Turtle Party – Multi-Agent Following Simulation (ROS 2, Python)

ROS-2 Turtlesim Python

This project simulates a chain of turtles following each other using ROS 2 and the turtlesim package. I created a ROS 2 Python package with custom nodes: one broadcaster to publish the pose of the lead turtle, and a follow node for each subsequent turtle to track its predecessor using real-time pose data. The project involved full setup of a ROS 2 workspace, writing nodes for frame transformations with tf2_ros, and configuring a launch file to spawn turtles and run all nodes in sync. The first turtle was controlled manually, while each following turtle dynamically adjusted its motion based on the one ahead. This task deepened my understanding of ROS 2 communication, tf broadcasting, multi-agent coordination, and workspace setup.
